# 产品介绍
针对视障人士乘坐公交车的障碍，包括：无法分辨多辆进站公交，实时公交信息不普及/失灵，缺少志愿者/工作人员等问题，此产品通过CV监测和识别算法，为用户提供稳定，准确的公交车线路的播报及引导。
